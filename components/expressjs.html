Installing

Step 1: Create a directory
mkdir exampleapp

Step 2: Go to the directory
cd exampleapp

Step 3: Open VS Code
code .

Step 4: Create a package.json
npm init or npm init -y

Step 5: Install Express
npm install express

Step 6: Install nodemon (automatically restart the node application)
npm install -g nodemon or npm install --save-dev nodemon

Usage

Step 1: Create an index.js in root directory
index.js

Step 2: inside -> index.js
const express = require('express');
const app = express(); const port = 500;

app.get('/', (req, res) => res.send('Hello World!'));

app.listen(port, () => console.log(`Server is running from port: ${port}`));

Step 3: Inside package.json
"scripts" : { "dev": "nodemon index.js"}

Step 4: Run local server
npm run dev
